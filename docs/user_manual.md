# パートナー要員管理システム - ユーザーマニュアル

## 1. はじめに

このマニュアルは、パートナー要員管理システムの使用方法について説明します。本システムは、パートナー会社の情報管理、案件管理、要員管理、応募管理などの機能を提供します。

## 2. システム要件

- ウェブブラウザ: Google Chrome, Firefox, Safari, Edge（最新版推奨）
- インターネット接続

## 3. システムへのアクセス

1. ブラウザを開き、システムのURLにアクセスします。
2. ログイン画面が表示されます。

## 4. ログイン方法

1. ユーザー名とパスワードを入力します。
2. 「ログイン」ボタンをクリックします。
3. 認証に成功すると、ダッシュボード画面に遷移します。

## 5. 画面構成

システムは以下の主要画面で構成されています：

- ダッシュボード
- パートナー会社一覧・詳細
- 案件一覧・詳細
- 要員一覧・詳細
- 応募一覧・詳細

各画面の左側にはナビゲーションメニューがあり、画面間を移動することができます。

## 6. パートナー会社管理

### 6.1 パートナー会社一覧の表示

1. ナビゲーションメニューから「パートナー会社」をクリックします。
2. パートナー会社一覧画面が表示されます。
3. 検索ボックスを使用して、特定のパートナー会社を検索できます。

### 6.2 パートナー会社の新規登録

1. パートナー会社一覧画面で「新規登録」ボタンをクリックします。
2. 登録フォームが表示されます。
3. 必要な情報（会社名、住所、電話番号、代表者名など）を入力します。
4. 「登録」ボタンをクリックして保存します。

### 6.3 パートナー会社の詳細表示

1. パートナー会社一覧画面で、詳細を表示したいパートナー会社の行をクリックします。
2. パートナー会社詳細画面が表示されます。
3. 基本情報と所属要員の一覧が表示されます。

### 6.4 パートナー会社情報の編集

1. パートナー会社詳細画面で「編集」ボタンをクリックします。
2. 編集フォームが表示されます。
3. 情報を更新します。
4. 「保存」ボタンをクリックして変更を保存します。

### 6.5 パートナー会社の削除

1. パートナー会社詳細画面で「削除」ボタンをクリックします。
2. 確認ダイアログが表示されます。
3. 「はい」をクリックして削除を確定します。

## 10. ダッシュボード

### 10.2 ダッシュボードの内容

- ステータス別の要員数グラフ

## 11. トラブルシューティング

### 11.1 ログインできない場合

- ユーザー名とパスワードが正しいか確認してください。
- Caps Lockキーがオンになっていないか確認してください。
- ブラウザのCookieが有効になっているか確認してください。
- システム管理者に連絡してアカウントのステータスを確認してもらってください。

### 11.2 画面が正しく表示されない場合

- ブラウザを最新バージョンに更新してください。
- ブラウザのキャッシュをクリアしてください。
- 別のブラウザで試してみてください。

### 11.3 データが保存されない場合

- 必須項目がすべて入力されているか確認してください。
- 入力値が正しい形式か確認してください。
- ネットワーク接続が安定しているか確認してください。
